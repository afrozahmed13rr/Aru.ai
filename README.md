Aru AI 🤖

Aru is my personal AI companion—a chill, witty, and caring friend.
About

This project is an AI-powered assistant designed to be more than just a chatbot. It features:

    Personal Memory: Remembers our conversations and thoughts.

    Hinglish Support: Communicates in a natural, casual mix of Hindi and English.

    FastAPI Backend: Runs as a web service, making it accessible from anywhere.

Tech Stack

    Language: Python

    Framework: FastAPI

    AI Engine: G4F (GPT-4o)

    Hosting: Render.com
